### 简介
适配器模式用于将一个类的接口转换成客户希望的另外一个接口。适配器模式使得原本由于接口不兼容而不能一起工作的类，可以一起工作。  
适配器模式分为：**类适配器** 和 **对象适配器**。由于JAVA等语言不支持*多继承*，所以经常使用的是**对象适配器**。  

---

### 角色

* Target：  
客户希望的目标接口，可以是抽象的，也可以是具体的类
* Adaptee：  
要被适配的类
* Adapter：
通过封装一个需要适配的类的对象，将原接口转换成目标接口
